# VB-Auto-v2.0


VB Auto is a polished vehicle inventory and sales management system built with VB.NET.
I designed it to simulate a real dealership workflow, from managing inventory to generating customer-ready receipts.

Overview

VB Auto Center is a desktop application that demonstrates:

Clean UI design principles
Structured application architecture
Real-world business logic (tax, add-ons, trade-ins)
Persistent data storage
Print-ready output generation

This project focuses on writing maintainable, scalable code not just making something that works.
